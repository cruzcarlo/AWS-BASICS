# AWS-BASICS
# AWS EC2 + Nginx Lab

# What We Did
This project shows the basic steps to set up Nginx on an AWS EC2 Free Tier instance and verify it in the browser.

# Steps
1. **Update All Packages**  
   Ran the system update to make sure everything is current.

2. **Installing Nginx**  
   Installed the Nginx web server on Amazon Linux 2023.

3. **Start the Nginx Service**  
   Started and enabled the Nginx service so it runs automatically.

4. **Put a Simple Header**  
   Created a custom `index.html` file with a simple header message.

5. **Result in Browser**  
   Opened the EC2 Public IPv4 in a browser and confirmed the page is served.
